Inventory
Info

This is your inventory. Your inventory is used to handle your items. You will see the item name on the left and the number of items of that type you have next to it.

On the right-hand side you will see three buttons.

  • Use
  • Drop
  • Destroy
Use allows you to use or consume that item. It is dependent on the item configuration.

Drop allows you to drop that item on the ground. It can be picked up again.

Destroy will destroy a single instance of that item. It cannot be recovered.
